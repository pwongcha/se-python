# Changes
(*changes*)

## Overview

Manage changes to an enrollment's network deployment.

### Available Operations

* [get_enrollment_change](#get_enrollment_change) - Get change status
* [delete_enrollment_change](#delete_enrollment_change) - Cancel a change
* [get_change_deployment_schedule](#get_change_deployment_schedule) - Get a deployment schedule
* [get_change_allowed_input_param](#get_change_allowed_input_param) - Get a change
* [post_change_allowed_input_param](#post_change_allowed_input_param) - Update a change
* [get_history_changes](#get_history_changes) - Get change history

See documentation for this operation in Akamai's Certificate Provisioning System API
<https://techdocs.akamai.com/cps/reference/post-change-allowed-input-param>

### Example Usage

```python

from aktest_se import Se

s = Se()

    "allowed_input_type_param": aktest_se.PathParamAllowedInputTypeParam.THIRD_PARTY_CERT_AND_TRUST_CHAIN,

    "enrollment_id": 10000,
    "account_switch_key": "1-5C0YLB:1-8BYUX",
    "request_body": {
        "certificates_and_trust_chains": [
            {
                "certificate": "-----BEGIN CERTIFICATE-----\n" +

                "-----END CERTIFICATE-----",
                "key_algorithm": aktest_se.KeyAlgorithm.ECDSA,
                "trust_chain": "-----BEGIN CERTIFICATE-----\n" +
                "MIID ... 4p5mA=\n" +
                "-----END CERTIFICATE-----\n" +
                "-----BEGIN CERTIFICATE-----\n" +
                "MIIDQ ... JuAIQ=\n" +
                "-----END CERTIFICATE-----",
            },

                "certificate": "-----BEGIN CERTIFICATE-----\n" +

                "key_algorithm": aktest_se.KeyAlgorithm.RSA,

                "MIID ... mA45p=\n" +
                "-----END CERTIFICATE-----\n" +
                "-----BEGIN CERTIFICATE-----\n" +
```
