# GetDeploymentsDeploymentsResponseSignatureAlgorithm

Indicates the SHA (Secure Hash Algorithm) function. You can use either `SHA-1` for a 160-bit (20-byte) hash or `SHA-256` for a 256-bit (32-byte) hash.


## Values

| Name      | Value     |
| --------- | --------- |
| `SHA_1`   | SHA-1     |
| `SHA_256` | SHA-256   |
