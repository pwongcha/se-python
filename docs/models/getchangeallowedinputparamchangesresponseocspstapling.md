# GetChangeAllowedInputParamChangesResponseOcspStapling

Indicates the OCSP stapling setting for the deployment. Use `on` to enable OCSP stapling, `off` to disable it, or `not-set` to ignore it.


## Values

| Name      | Value     |
| --------- | --------- |
| `ON`      | on        |
| `OFF`     | off       |
| `NOT_SET` | not-set   |
