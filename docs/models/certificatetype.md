# CertificateType

Either `san`, `single`, `wildcard`, `wildcard-san`, or `third-party`. See [Enrollment.validationType Values](#validationtype) for details.


## Values

| Name           | Value          |
| -------------- | -------------- |
| `SAN`          | san            |
| `SINGLE`       | single         |
| `WILDCARD`     | wildcard       |
| `WILDCARD_SAN` | wildcard-san   |
| `THIRD_PARTY`  | third-party    |
